    Kaavoja

    SUURE KAAVA
    Insinöörijännitys σ (σe) = F / A0
    Todellinen jännitys σt = F / A = σe(1 + εe)
    Insinöörivenymä ε (εe) = Δl / l0
    Todellinen venymä εt = ln(1 + εe)
    Kimmokerroin (Youngin moduuli) E = σ / ε
    Leikkausmoduuli (Liuku-) G = τ / γ = E / 2(1 + ν)
    Bulkkimoduuli K = E / 3(1 - 2ν)
    Poissonin vakio ν = -εx / εz = -εy / εz
    Schmidin laki (m=orientaatiotekijä) τcrss = σy / m
    Hall-Petch yhtälö (raekoon [d] vaikutus lujuuteen) σy = σ0 + ky / sqrt(d)

    Päätteitä: -LD (matalatiheyksinen), -LLD (lineaarinen matalatiheyksinen), -HD (korkeatiheyksinen),
